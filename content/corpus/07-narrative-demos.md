# 叙事解谜 Demo 合集：离线档案馆

除了正式项目，他还做了一批网页像素叙事解谜 Demo，统一收在「离线档案馆 / OFFLINE ARCHIVE」合集里，线上入口 arcade.remi.run。六个 Demo 气质一致：都是近未来视角下，从一堆旧数据（聊天备份、客服工单、旧手机、失效的个人主页）里，把一个人重新拼回来。纯前端、零依赖或极轻构建，单周目 10–25 分钟。

六个作品：

- **低算力时代恋爱样本**：2036 年的研究员，分析人类用户与早期 AI 助手 Remi 的聊天备份，为对话打标签、恢复被抹除的异常片段。源码 github.com/jane-uske/lce-love-sample。
- **最后一个 QQ 群**：从一个停在过去的怀旧 QQ 群的公告、合照、歌单里，翻出一群人十几年没说出口的告别。github.com/jane-uske/last-qq-group。
- **人类复古博物馆：2026 展区**：未来博物馆的临时策展员，整理「2026 手动生活时代」展区，修正系统误读的标签，拼出一位普通人的生活痕迹。github.com/jane-uske/retro-museum-2026。
- **最后一个人工客服**：2049 年的审计员复查 2026 年的电商客服工单备份，样本编号 H-0715——平台最后一批人工客服之一。github.com/jane-uske/last-human-support。
- **404 旅馆：最后一次请求**：失效个人主页的源码里藏着一家旅馆，每个房间对应一个 HTTP 状态码，去找回一个被世界遗忘的人。github.com/jane-uske/hotel-404。
- **旧手机里的她**：旧货市场捡到一台还有电的旧手机，通过短信、相册、备忘录和一个损坏的 AI 助手拼出旧主人的故事。技术上是唯一用 React + TypeScript + Vite 的一个，像素画为字符网格渲染的内联 SVG、音效为 WebAudio 合成，无任何外部图片/字体/音频资源。github.com/jane-uske/old-phone-demo。

这批 Demo 体现的是他把「一个完整可玩的叙事产品」从 0 做到上线的能力：叙事设计、状态机、演出序列、存档与防呆分支、以及零依赖工程化，全链路一人完成。
